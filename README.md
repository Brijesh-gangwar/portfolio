## Environment setup

1) Install or ensure you have JDK 17+ â€” required for Android and language server features.

macOS (Homebrew):
```bash
brew install openjdk@17
sudo ln -sfn $(/usr/libexec/java_home -v 17) /Library/Java/JavaVirtualMachines/jdk-17.jdk
```

2) Install Flutter and add it to PATH. Example:
```bash
# assuming you clone Flutter into ~/tools/flutter
export PATH="$HOME/tools/flutter/bin:$PATH"
```

3) Add to your shell rc (.zshrc or .zprofile) and restart terminal.

4) Restart VS Code, run `flutter doctor` and ensure all required tools are installed.
